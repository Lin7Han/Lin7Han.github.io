虚幻引擎知识库

这是一个面向虚幻引擎开发者（以程序工种为主）的知识库，其中的文章从源码层面剖析引擎各个重要模块的工作原理，帮助读者深入理解引擎底层的实现细节，从而更好地进行游戏/框架的开发和引擎的定制。

信息

虚幻引擎知识库不面向初学者，假定读者已经掌握以下技能：

  • 熟悉标准 C++ 的常用语言特性，熟悉虚幻引擎 C++ 与标准 C++ 的主要差异。
  • 熟悉标准 STL 或虚幻引擎 STL 中的至少一个。
  • 熟悉虚幻引擎的架构设计、常用模块和源代码结构。

以下技能不要求掌握，但可以帮助读者更快理解内容：

  • 熟悉并理解主流的游戏引擎架构模式、常见的设计模式和设计原则。
  • 熟悉虚幻引擎的编码风格和常用术语。
  • C++ 或虚幻引擎实践经验。
