虚幻引擎知识库

这是一个面向虚幻引擎开发者（以技术人员为主）的知识库，其中的文章从源码层面剖析引擎各个重要模块的工作原理，帮助读者深入理解引擎底层的实现细节，从而更好地进行游戏/框架的开发和引擎的定制。

虚幻引擎知识库分为基础和进阶两个板块。划分依据是模块在引擎中所处的地位，而非难度： 基础板块中的内容是引擎的核心组件，没有它们引擎就无法运行；进阶板块中的内容是基于核心组件构建的上层功能组件，专注于实现某个特定功能，但并不是构成引擎的关键要素，甚至会收录一些热门程度非常高的第三方组件。

虽然划分依据不是难度，但两个板块的难度分布大体呈现为：基础板块本身可能非常复杂，但它要求的前置知识很少；进阶板块本身可能并不复杂，但它要求读者先理解基础板块的内容。 虽然这个学习曲线可能比较令人难以接受，但探讨原理确实不像工程实践一样可以先抛开底层不谈。 如果读者不理解前置知识的话，也就很可能无法有效理解接下来的内容，最后的学习效率反而不如按部就班。 因此，推荐的阅读顺序有两种：

  1. 如果对某项特定内容感兴趣，可以从关注点开始递归阅读，每篇文章的头部会标注前置知识。
  2. 如果没有特定目标，顺序阅读即可。

信息

虚幻引擎知识库不面向初学者，假定读者已经掌握以下技能：

  • 熟悉标准 C++ 的常用语言特性，熟悉虚幻引擎 C++ 与标准 C++ 的主要差异。
  • 熟悉标准 STL 或虚幻引擎 STL 中的至少一个。
  • 熟悉虚幻引擎的架构设计、常用模块和源代码结构。
