Class Flags

引擎版本：5.4更新日期：2024-09-29状态：In Progress

目录​

标签分类简介
AbstractTraits该类型是抽象类，不能实例化。
ConstTraits该类型是不可变的，主要影响蓝图的行为。
IntrinsicTraits该类型是没有 UHT 模板代码的内建类型。
NativeTraits该类型是由程序定义的 Native 类型。
CompiledFromBlueprintTraits该类型从蓝图编译而来。
DefaultToInstancedTraits所有引用该类型或其子类的属性都被视为 Instanced。
HasInstancedReferenceTraits该类型包含 Instanced 属性。
NotPlaceableTraits该类型的 Actor 不能在编辑器中直接实例化。
DeprecatedTraits该类型已废弃，不应该使用且不会参与序列化。
InterfaceTraits该类型是一个接口类，只能用于 NoExportTypes.h 中定义的引擎内建类型，其他类型应该用 UINTERFACE() 宏。
OptionalTraits该类型只在特定上下文中存在，多用于 Editor-Only 的类型。
MinimalAPITraits该类型只导出最小信息（反射信息）。
RequiredAPITraits该类型会导出所有成员，在类声明前标记 API 宏即为 RequiredAPI，与 MinimalAPI 互斥。
NeedsDeferredDependencyLoadingTraits该类型需要延迟加载依赖，多用于有循环依赖的蓝图类。
TokenStreamAssembledLifecycle该类型的类对象已经调用了 AssembleReferenceTokenStream()，可以对该类型的实例进行可达性分析。
ReplicationDataIsSetUpLifecycle该类型的类对象已经调用了 SetUpRuntimeReplicationData()，可以对该类型的实例进行网络同步。
ConstructedLifecycle该类型的类对象已经构建完成，可以正常使用，但在热重载时可能处于旧版本。
NewerVersionExistsLifecycle该类型正在参与蓝图编译，且有新版本。
TransientSerialization该类型的实例不会保存到资产中。
MatchedSerializersSerialization该类型可以使用 FStructuredArchive 进行序列化，只能用于 NoExportTypes.h 中定义的引擎内建类型。
ConfigConfig该类型的 CDO 实例化时需要加载配置信息，其他实例从 CDO 拷贝配置信息。
PerObjectConfigConfig该类型的所有实例都需要加载配置信息。
DefaultConfigConfig该类型的配置信息只存在于 Default*.ini 中，需要和 Config 同时存在。
PerPlatformConfigConfig该类型的配置信息会被各平台的 ini 覆盖，切换预览平台时需要重新加载配置。
GlobalUserConfigConfig该类型的配置信息存在于全局的 User*.ini 中。
ProjectUserConfigConfig该类型的配置信息存在于项目的 User*.ini 中。
ConfigDoNotCheckDefaultsConfig该类型在保存配置信息时不检查是否与父类的值相同。
EditInlineNewEditor该类型可以在编辑器中直接实例化。
CollapseCategoriesEditor该类型的属性在编辑器中不按 Category 排列，而是直接平铺开展示。
HiddenEditor该类型不在编辑器中显示。
HideDropDownEditor行为与 Hidden 类似，具体取决于使用方判断的是哪个标签。
